# ❓ 常见问题

NexusBot 使用过程中的常见问题解答。

---

## 安装问题

### Q: Windows 安装时提示"无法验证发布者"？
**A**: 这是正常的，点击"仍要运行"即可。或者右键 → 属性 → 解除锁定。

### Q: macOS 提示"无法打开，因为无法验证开发者"？
**A**: 系统偏好设置 → 安全性与隐私 → 点击"仍要打开"。

### Q: Linux 安装后无法启动？
**A**: 检查依赖：`sudo apt-get install -f`

## 配置问题

### Q: 如何获取钉钉 AppKey？
**A**: 参考 [钉钉配置指南](./platforms/dingtalk.md)

### Q: 本地模型如何选择？
**A**: 
- 4GB 内存：3B 模型
- 8GB 内存：7B 模型
- 16GB+ 内存：13B+ 模型

### Q: API Key 在哪里配置？
**A**: 设置 → AI 模型 → 输入 API Key

---

## 使用问题

### Q: 机器人不回复消息？
**A**: 检查清单：
1. 服务是否运行
2. AI 模型是否配置正确
3. 消息平台配置是否正确
4. 查看日志文件

### Q: 如何安装技能？
**A**: 技能市场 → 搜索 → 安装

### Q: 如何创建工作流？
**A**: 工作流 → 新建 → 拖拽节点 → 保存

---

## 性能问题

### Q: 响应速度慢？
**A**: 
- 使用本地模型
- 升级硬件
- 减少并发请求

### Q: 内存占用高？
**A**: 
- 关闭不用的技能
- 清理缓存
- 重启服务
