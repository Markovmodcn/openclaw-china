# 🚀 NexusBot 快速开始

5 分钟快速上手 NexusBot！

---

## 第一步：启动 NexusBot

安装完成后，双击桌面图标启动 NexusBot。

---

## 第二步：选择 AI 模型

### 选项 A：使用本地模型（推荐）

**优势**：完全免费、隐私安全、无限制使用

1. **安装 Ollama**
   ```bash
   # Windows/macOS/Linux
   访问 https://ollama.com 下载安装
   ```

2. **下载模型**
   ```bash
   ollama pull qwen2.5:7b
   ```

3. **在 NexusBot 中配置**
   - 选择"本地模型"
   - 选择 Ollama
   - 测试连接

### 选项 B：使用在线模型

1. **选择 AI 服务商**
   - DeepSeek（推荐，性价比高）
   - Kimi（长文本处理）
   - 通义千问（阿里云）

2. **获取 API Key**
   - 访问服务商官网注册
   - 获取 API Key

3. **在 NexusBot 中配置**
   - 选择"在线模型"
   - 输入 API Key
   - 测试连接

## 第四步：开始对话

配置完成后：

1. 在消息平台中找到你的机器人
2. 发送消息测试
3. 机器人会自动回复

**示例对话**：
```
你：你好
机器人：你好！我是 NexusBot，有什么可以帮助你的吗？

你：帮我总结一下今天的新闻
机器人：[开始搜索并总结新闻...]
```

---

## 第五步：安装技能插件

1. **打开技能市场**
   - 点击左侧菜单"技能市场"

2. **浏览技能**
   - 搜索你需要的功能
   - 查看技能详情和评分

3. **一键安装**
   - 点击"安装"按钮
   - 等待安装完成

4. **使用技能**
   - 在对话中直接使用
   - 例如：`@GitHub 查看我的仓库`

## 💡 使用技巧

1. **善用 @提及**
   - 在群聊中 @机器人
   - 私聊直接发送

2. **使用技能**
   - `@技能名 命令`
   - 例如：`@GitHub 搜索 nexusbot`

3. **查看帮助**
   - 发送 `帮助` 查看可用命令
   - 发送 `技能列表` 查看已安装技能
