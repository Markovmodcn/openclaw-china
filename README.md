## 项目简介

NEUXSBOT 是基于 OpenClaw 深度优化的中国版本，核心方向是把 AI 从被动问答升级为主动执行。  
产品以桌面端为执行核心，结合图形化配置、技能系统与多模型协同能力，面向真实生产场景。

## 与本地网页一致的导航结构

以下内容与本地前端导航保持一致：

- 首页
- 产品
- 订阅
- 定制
- 下载
- 文档

产品下拉菜单与本地页面一致：

- `NEUXSBOT IDE`：AI 自主开发，全自动编程
- `NBPHONE`：AI 原生终端与生态入口
- `数据分析系统`：AI 股票研判与策略平台

---

## IDE 开发代码能力（重点）

- 内置 `NEUXSBOT IDE`，面向真实开发流程
- 支持从需求描述到代码生成、修改、迭代的连续协作
- 支持多轮上下文、任务历史与技能增强开发
- 支持前端开发、自动化脚本、测试脚本等常见代码任务

---

## 图形化一键安装

- 提供图形化使用入口，降低部署门槛
- 下载页支持 Windows / macOS / Linux
- 提供命令行快速安装方式（可选）
- 文档中心提供快速开始、模型配置、工作流说明

---

## 核心能力

- AI 操作系统体验：从“对话”到“执行”
- 多模型协同：支持云端与本地模型切换
- 技能系统：覆盖开发、文档、自动化等场景
- 工作流能力：任务分解、执行、反馈闭环
- 数据分析子系统：多维概率模型分析实验室
