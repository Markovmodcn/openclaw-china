## 项目定位 | Positioning

NEUXSBOT 是基于 OpenClaw 深度优化的中国版本，核心目标是让 AI 从“被动问答”升级为“主动执行”。  
它以桌面端为执行核心，提供图形化配置、技能系统、多模型协同与自动化工作流能力。

NEUXSBOT is a China-focused edition built on top of OpenClaw, designed to move AI from passive chat to active execution.  
It provides a GUI-first desktop workflow with skills, multi-model orchestration, and automation.

---

## 核心特色 | Key Features

### 1) IDE 开发代码能力（重点） | IDE for Code Development

- 内置 `NEUXSBOT IDE`，面向真实开发流程
- 支持从需求描述到代码生成、修改、迭代的连续协作
- 支持多轮上下文、任务历史、技能增强开发
- 适合前端、自动化脚本、测试脚本等代码场景

- Built-in `NEUXSBOT IDE` for real coding workflows
- Continuous collaboration from prompt -> code -> refine
- Multi-turn context, task history, and skill-augmented development
- Suitable for frontend, scripts, and test automation workflows

### 2) 图形化一键安装 | GUI One-Click Installation

- 提供图形化使用入口，降低部署与使用门槛
- 支持 Windows / macOS / Linux 版本下载
- 提供命令行快速安装选项（可选）

- GUI-first usage with a lower setup threshold
- Download support for Windows / macOS / Linux
- Optional command-line quick install

### 3) OpenClaw 中国版体验 | OpenClaw Chinese Edition Experience

- 中文文档与中文场景适配
- 支持国内常用协作沟通场景（企业方向）
- 模型配置、工作流配置、技能配置可视化

- Chinese-first docs and localized workflows
- Enterprise communication scenario adaptation
- Visual configuration for models, workflows, and skills

### 4) 多模型与技能系统 | Multi-Model + Skills

- 支持多模型切换（云端模型与本地模型）
- 支持本地模型（如 Ollama）与隐私优先部署方式
- 技能系统可扩展，覆盖开发、文档、自动化等场景

- Multi-model switching across cloud and local models
- Local model support (e.g., Ollama) for privacy-first usage
- Extensible skill system for coding, docs, and automation

### 5) 数据分析子系统 | Data Analysis Subsystem

- 提供“多维概率模型分析实验室”
- 融合马尔可夫链、LSTM、XGBoost 等模型思路
- 支持多分析模块与历史回测验证流程

- Includes a multi-dimensional probability analysis lab
- Combines Markov chain, LSTM, and XGBoost ideas
- Supports multiple analyzers with backtesting flows

## 前端模块概览 | Web Modules Overview

以下模块来自本地前端路由结构（`C:\markovbot\markovbot\web\frontend`）：

| 模块 Module | 路由 Route | 用途 |
|---|---|---|
| 首页 Home | `/` | AI 操作系统入口与能力总览 |
| IDE | `/ide` | 代码开发与任务执行工作区 |
| NBPHONE | `/product/phone` | 终端产品方向介绍 |
| 数据分析 | `/data-analysis` | 多模型分析实验室 |
| 文档中心 | `/docs` | 快速开始、模型配置、工作流指南 |
| 下载页 | `/download` | 多平台安装入口 |
| 企业版 | `/enterprise` | 企业协作与私有化方向 |
| 用户中心 | `/member` | 账号与控制台功能 |
